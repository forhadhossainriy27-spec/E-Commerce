# 🛒 Laravel E-Commerce

A modern, scalable, and production-ready E-Commerce web application built with **Laravel 13**, **Blade**, and **Tailwind CSS**.

> This project is being developed step by step using Git. Each completed feature is committed separately to keep the development history clean and easy to follow.

---

## 🚀 Tech Stack

- **Framework:** Laravel 13
- **Frontend:** Blade + Tailwind CSS
- **Build Tool:** Vite
- **Database:** MySQL
- **Authentication:** Laravel Breeze
- **Version Control:** Git & GitHub

## ⚙️ Installation

Clone the repository:

```bash
git clone <repository-url>
```

Go to the project directory:

```bash
cd ecommerce
```

Install dependencies:

```bash
composer install
npm install
```

Copy the environment file:

```bash
cp .env.example .env
```

Generate the application key:

```bash
php artisan key:generate
```

Configure your database inside the `.env` file.

Run migrations:

```bash
php artisan migrate
```

Start the development server:

```bash
php artisan serve
```

Run Vite:

```bash
npm run dev
```
